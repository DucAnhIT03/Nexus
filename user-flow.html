📋 Tóm tắt User Flow – NexusMarket

🔐 Luồng xác thực

  1. Người dùng mới → Đăng ký (họ tên, email, MK)
  2. Người dùng cũ → Đăng nhập (email + MK)
  3. Quên MK → Reset qua email
  4. Thành công → Chuyển về Trang chủ

🛍️ Luồng khám phá

  1. Trang chủ → xem banner, danh mục, SP nổi bật
  2. Chọn danh mục → Marketplace (lọc, tìm kiếm)
  3. Click sản phẩm → Chi tiết SP
  4. Xem ảnh, giá, mô tả, seller info, đánh giá

💳 Luồng mua hàng

  1. Chi tiết SP → "Mua ngay" → Thanh toán
  2. Hoặc: "Thêm vào giỏ" → Giỏ hàng → Thanh toán
  3. Hoặc: "Yêu thích" → Wishlist → Giỏ hàng
  4. Thanh toán xong → Đơn hàng → SP đã mua

👤 Luồng tài khoản & Seller

  1. Navbar avatar → Hồ sơ (chỉnh sửa info, bảo mật)
  2. Navbar 🔔 → Thông báo (đơn hàng, KM)
  3. Click "Trở thành Seller" → Seller Dashboard
  4. Seller: quản lý SP, theo dõi doanh thu, xử lý đơn

🤖 Gemini AI Assistant

Floating chatbox hiển thị ở mọi trang (trừ Login/Register). Hỗ trợ hỏi đáp, tư vấn sản phẩm, giải đáp thắc mắc thời gian thực.
